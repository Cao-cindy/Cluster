The input/output and function of each Python script are as follows:

class_cluster2018_an_drug: Categorizing antibiotics into groups based on known drug information and assigning adjuvants to these groups using Prism II algorithms. The inputs are the interactions between antibiotics and antibiotics, antibiotics and adjuvants, and adjuvants and adjuvants. The output is the clustering results.

cluster2018_an: Clustering antibiotics using the Prism II algorithm. The inputs are antibiotics and antibiotic interactions. The output is the antibiotic clustering results.

cluster2018_an_drug: Clustering antibiotics and adjuvants using the Prism II algorithm. The inputs are the interactions between antibiotics and antibiotics, antibiotics and adjuvants, and adjuvants and adjuvants. The output is the clustering results.
